# ServiceFlow

ServiceFlow is a small Express and EJS service directory website for digital services, AI automation, and business growth enquiries.

## Features

- Home page
- Services page
- Service detail pages
- Contact enquiry form
- Server-side contact form validation
- Enquiry saving to a local JSON file

## Tech Stack

- Node.js
- Express
- EJS
- Express EJS Layouts
- Bootstrap
- Custom CSS
